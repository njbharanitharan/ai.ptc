About This Project

The AI Plant Tissue Culture Assistant + Live Wiki Search is a fully interactive web tool designed to support plant biotechnology students, researchers, and lab technicians. It combines two powerful modules — an AI-powered protocol generator and a real‑time Wikipedia lookup engine — inside a visually immersive forest‑themed interface.

How We Built It

This project is the result of several weeks of brainstorming, research, prototyping, and refinement. Our goal was not just to build another informational website, but to create a tool that feels alive — a digital extension of an actual biotechnology laboratory. Every component, from UI layout to data processing flow, was designed to simulate the experience of working with real plant tissue culture protocols.

Below is an expanded breakdown of the full technical and conceptual foundation that shaped the development of the AI Plant Tissue Culture Assistant.

1. Front‑End UI & Design

The interface was carefully crafted to feel clean, professional, and laboratory‑inspired. We used pure HTML, CSS, and JavaScript to maximize performance and reduce dependency on heavy frameworks. Some additional details include:

  • A fully responsive column system that adjusts intelligently from widescreen desktops to compact mobile devices.
  • A polished laboratory background image chosen to resemble a sterile, well‑lit real‑world biotech environment.
  • Micro‑interactions such as button glows, hover states, subtle shadows, and sliding transitions to give a sense of physical depth.
  • Typography optimized for readability, using the Poppins typeface for a modern, scientific aesthetic.
  • Rounded panel geometry (16–22px radius) inspired by modern scientific software dashboards.
‑End UI & Design

The user interface was developed using pure HTML, CSS, and JavaScript, without any external frameworks. Key features include:

  • A multi‑layered 3D parallax forest background with mountains, trees, fog, and fireflies.
  • Glass‑blur panel design using backdrop-filter to create a futuristic biotech feel.
  • A split, responsive layout for the AI Protocol Panel and Wiki Search Panel.
  • Dynamic animations, glowing borders, and forest‑inspired gradients.

2. AI Protocol Engine

At the core of the system is an intelligent protocol retrieval and formatting engine. While not a full AI model, it behaves like one through careful structuring of data and logic. Expanded features include:

  • Natural‑language keyword recognition that allows flexible search terms such as common names, scientific names, or protocol types.
  • Adaptive protocol formatting that restructures stored data into clean, step-by-step laboratory instructions.
  • Contextual metadata extraction used to automatically insert sterilization times, hormone concentrations, and media formulations.
  • A custom typewriter animation engine that simulates digital lab notebook output.
  • Batch update ability to integrate new plants into the system without modifying core code.

The combination of JSON-based datasets and JavaScript parsing allows the tool to cover a wide range of species and experimental conditions.

3. Live Plant Search

Live Plant Search

The right‑side panel connects to the official INTERNET REST API to fetch:

  • Plant descriptions
  • Scientific summaries
  • Thumbnail images
  • Automated speech output using our custom reader

4. Extra Features

  • Firefly particle animation created using JavaScript loops
  • Auto‑generated audio narration using Web Speech API
  • Error‑handling for missing data and failed API requests
  • Mobile‑adaptive layout for clean viewing on any device

Our Vision

Our mission is to make plant biotechnology more accessible by combining AI, interactive visuals, and clean UI engineering. By merging real lab knowledge with fast information access, this tool supports both beginners and advanced researchers.
